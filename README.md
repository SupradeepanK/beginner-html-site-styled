# 🌐 Beginner HTML Styled Site (DevOps Integrated)

## 📌 Overview

This project is a **static website built using HTML and CSS**, originally based on a beginner-friendly learning template and enhanced for use in a **DevOps CI/CD pipeline**.

It demonstrates how a simple frontend application can be **containerized and deployed using modern DevOps practices**.

> This project is adapted from learning resources similar to those provided by MDN Web Docs for HTML and CSS basics.

---

## 🚀 Features

* Single-page responsive website
* Styled using CSS for improved UI
* Beginner-friendly structure and design
* Lightweight and fast-loading static site

---

## ⚙️ DevOps Integration

This project is part of a complete DevOps workflow:

1. Code pushed to GitHub
2. Jenkins pipeline triggers automatically
3. Docker image is built
4. Static site is deployed using Nginx container
5. Application is hosted on AWS EC2
